# FoodMate

**FoodMate**는 UI만 구성한 프로젝트입니다. 현재 로그인, 회원가입, 온보딩, 홈 화면, 푸드메이트, 레시피 검색, 채팅, 마이페이지 UI가 구현되어 있습니다.
