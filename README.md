# FoodMate

**FoodMate**는 디자인구성 및 UI를 개발한 프로젝트입니다. 로그인, 회원가입, 온보딩, 홈 화면, 채팅, 마이페이지 및 푸드메이트 관련 UI가 구현되어 있습니다.
